# Contributing to ElectionEdu

Thank you for your interest in contributing to ElectionEdu! We welcome contributions that improve the accuracy, accessibility, and quality of our election education platform.

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Install dependencies** using `npm install`.
3. **Make your changes**. Ensure your code follows the established style guide (Prettier).
4. **Write tests** for any new logic using Vitest.
5. **Run existing tests** using `npm test` to ensure no regressions.
6. **Submit a Pull Request** with a clear description of your changes and why they are beneficial.

## Code Standards
- Use ES6+ features.
- Write JSDoc comments for all exported functions and modules.
- Ensure all interactive elements remain fully ARIA-compliant.
- Keep the UI responsive and performant.

## Reporting Issues
If you find a bug or have a feature request, please open an issue on the GitHub repository with detailed steps to reproduce or a clear description of the proposed feature.
